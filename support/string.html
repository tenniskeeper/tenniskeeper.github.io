What is Tennis String Tracking?

Are you still using one of these to track your tennis strings?


Now we have a better way to track your strings using your iPhone. Select the amount of hours you would like to track on a specific string and we will notify you once you have reached the playing hours. You track your playing hours by selecting the string you were using when you save your tennis activity. String tracking is available for singles, doubles, hitting and lesson activities.


How to enable String Tracking?

To enable String Tracking, first go to TennisKeeper > More > String Tracking

  1. Tap the (+) to add a string.
  2. Enter the information for your string, such as the number of playing hours you want to track.

Associate the string with a tennis activity

Next, when you add a new tennis activity, or update an existing tennis activity, you can select the string you use with that activity.


Get notified


That's all you need to do. TennisKeeper will track the playing hours and we will update you with a notification once you have reached the hours on the string.



Be sure to click on the notification to get your discount coupon code for your string purchase.
