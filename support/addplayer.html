How do I add a player?

  1. Go to TennisKeeper > Players.
  2. Tap the (+) button.
  3. Enter the player name and tap Done.

You should see the player added to the All Players list.

Now you can update the player profile by following the update player instructions.

The free edition only allows you to add 3 players. Upgrade to the Pro Edition to add unlimited number of players.
