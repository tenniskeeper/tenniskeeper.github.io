How do I change an Activity?

If you logged an activity type incorrectly, for example, if you recorded a match as Singles instead of Doubles, you can easily change it:

  1. Go to TennisKeeper > Activities.
  2. Swipe left on an existing Activity.
  3. Tap More. Then select Change Activity Type.
  4. Select the new Activity. Tap Change.
